프로그램명 : C#을 이용하여 간단한 1:N 비동기 채팅 프로그램

설명 : 1. MultiChatServer.exe 실행하여 서버를 먼저 구동한다
           - 포트 번호는 9000번
           - IP는서버의 IP를 그대로 사용
         2. MultiChatClient.exe 실행, 포트번호는 서버의 포트 번호를 입력후 연결버튼 처리
           - 클라이언트측 프로그램의 IP는 유동적

작업내용 : 참조사이트의 소스 활용하여 버그 수정 및 예외 처리, 기능개선 등을 처리함.
          

기타 설정 : 기존의 서버의 포터 사용유무를 판단 하여 포트번호를 설치할것
